Wireguard的简单配置

2024-05-25 · 更新于2024-11-15 · 1 分钟 · 192 字 · Kyxie
目录
  • Linux配置Wireguard
  • Windows配置wireguard

Linux配置Wireguard#

  • 下载wireguard

    sudo apt update
sudo apt upgrade
sudo apt install wireguard

  • 进入/etc/wireguard，并生成密钥对

    sudo -i
cd /etc/wireguard
wg genkey | tee privatekey | wg pubkey > publickey

  • 编辑配置文件wg0.conf

    [Interface]
PrivateKey = [LINUX_PRIVATE_KEY]
Address = 10.0.0.1/24
ListenPort = 51820

[Peer]
PublicKey = [WINDOWS_PUBLIC_KEY]
Endpoint = [WINDOWS_IP]:51820
AllowedIPs = 10.0.0.2/32

  • 启动wireguard

    sudo wg-quick up wg0

Windows配置wireguard#

  • 从这里下载wireguard app

  • Add Tunnel → Add empty tunnel

  • 编写配置文件

    [Interface]
PrivateKey = [WINDOWS_PRIVATE_KEY]
ListenPort = 51820
Address = 10.0.0.2/24

[Peer]
PublicKey = [LINUX_PUBLIC_KEY]
AllowedIPs = 10.0.0.0/24
Endpoint = [LINUX_IP]:51820

  • 点击运行
