使用U盘安装Debian Server

Header#

刚买了一台新的Mini PC打算作为自己的服务器，现在使用U盘引导它安装Debian。

需要准备#

  • 显示器
  • HDMI线连接显示器
  • 键盘
  • 至少8GB的U盘
  • 网线

装系统#

  • 官网下载Debian镜像
  • 准备一个至少8GB的U盘，安装Etcher，按照流程将Ubuntu镜像烧写到U盘中
  • 将U盘插入PC，启动，然后不断点击F7键，不同的主板按键不同，然后进入Bios
  • 我需要将整个磁盘都分配给Debian，后续也不折腾的话可以不选择LVM Group，这样只会产生两个分区，/和/boot/efi
  • 后面的无脑填就可以了，等到进入系统

修改IP地址#

  • 我打算将一个网口连接在路由器上，因此需要修改默认的IP地址

  • 首先ip addr查看当前的地址信息

  • 我们打算把enp0s1的IP地址修改为192.168.2.66，然后默认网关为主路由器192.168.2.1。我们需要编辑文件01-netcfg.yaml，没有的话就新建

    network:
  version: 2
  renderer: networkd
  ethernets:
    enp1s0:
      dhcp4: no
      addresses:
        - 192.168.2.66/24
      routes:
        - to: default
          via: 192.168.2.1
      nameservers:
        addresses:
          - 8.8.8.8
          - 8.8.4.4

  • 应用Netplan配置

    sudo netplan apply

  • 重启网络服务（可选）

    sudo systemctl restart systemd-networkd

  • 插入网线，验证配置

    ip addr

其他配置#

剩下的就是一些基本的其他配置了

  • 更换时区

    sudo timedatectl set-timezone America/Toronto

  • 安装Cockpit（端口9090）

    sudo apt install cockpit -y
sudo apt install cockpit-podman -y
sudo apt install cockpit-networkmanager -y
sudo apt install cockpit-storaged -y
sudo systemctl start cockpit
sudo systemctl enable cockpit
